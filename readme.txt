== Description ==

A plugin allowing to display tweets on the blog with options to filter by tags

== Installation ==

Upload the O2Tweet plugin to your blog, activate it and update the parameters.
You can update the tweets with a crontab instead of doing so while loading one of the blog's pages (which will slow your blog).
